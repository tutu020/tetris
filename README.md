# Galaxy Drop

de blauwe bal is het projectiel dat omlaag valt door gravity

de gele sterren zijn de targets --> raken verdwijnen = punten

idee van iemand: je bal komt weer naar boven zodra de bal in de mond valt (kai cenat)

Mechanics:

1. je schiet de bal van boven
2. gravity trekt 'm naar beneden
3. de bal stuitert tegen sterren --> sterren verdwijnen (met animatie)
4. doel van het level: alle sterren in zo min mogelijk schoten weghalen

Waarschijnlijk voegen:

Combo gravity: elke keer dat je een ster raakt, wordt gravity sterker (bal versnelt --> chaos)

Time slow zone: een lichtblauwe cirkel ergens: als de bal erin komt, gaat alles even slow-motion.
